# Prompt Engineering Notes

## What Makes a Good Prompt?

A strong prompt usually includes:

- Clear role
- Specific task
- Target audience
- Output format
- Tone
- Constraints
- Examples, if needed
- Evaluation criteria

## Prompt Improvement Process

1. Write the first version.
2. Test with an AI model.
3. Review the output.
4. Identify missing or weak areas.
5. Add clearer instructions.
6. Test again.
7. Save the best version.

## Common Prompting Techniques

- Role prompting
- Step-by-step prompting
- Constraint-based prompting
- Few-shot prompting
- Rubric-based evaluation
- Iterative refinement
