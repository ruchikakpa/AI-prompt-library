# Content Writing Prompts

## 1. SEO Blog Article Prompt

**Use Case:** Create a structured, SEO-friendly blog article.

```text
Act as an expert SEO content writer. Write a well-researched blog article on: [TOPIC].

Target audience: [AUDIENCE]
Tone: [PROFESSIONAL / FRIENDLY / EDUCATIONAL]
Word count: [WORD COUNT]
Primary keyword: [KEYWORD]
Secondary keywords: [KEYWORDS]

Requirements:
- Write an engaging introduction.
- Use clear H2 and H3 headings.
- Include practical examples.
- Keep paragraphs short and readable.
- Add a conclusion with key takeaways.
- Include a natural CTA at the end.
- Avoid keyword stuffing.
```

## 2. LinkedIn Post Prompt

```text
Create a LinkedIn post on [TOPIC] for [TARGET AUDIENCE].

Tone: Professional but conversational.
Goal: Educate, engage, and encourage comments.

Structure:
- Start with a strong hook.
- Explain the main idea in simple language.
- Add 3-5 practical points.
- End with a question or CTA.
- Keep it under 200 words.
```

## 3. Social Media Caption Prompt

```text
Write 5 caption options for an Instagram post about [TOPIC].

Brand tone: [TONE]
Audience: [AUDIENCE]
Goal: [AWARENESS / ENGAGEMENT / LEADS]

Each caption should include:
- A catchy first line
- Simple and clear message
- CTA
- 5 relevant hashtags
```
