# AI Prompt Library

A practical prompt engineering portfolio created by **Ruchika Rani** to showcase prompt design, AI content evaluation, LLM output assessment, research workflows, summarization, content creation, and productivity automation.

## About This Repository

This repository contains reusable prompt templates, prompt evaluation rubrics, and sample AI workflow case studies. It is designed for roles such as:

- AI Prompt Engineer
- AI Content Evaluator
- LLM Evaluator
- AI Trainer
- Generative AI Specialist
- AI Content Writer
- AI Quality Analyst

## How to Use

1. Select a prompt category from the `prompts` folder.
2. Copy the prompt and customize it for your task.
3. Use the evaluation rubrics to assess the AI output.
4. Record prompt versions and results in the testing log template.
5. Improve the prompt based on output quality.

## Disclaimer

This repository is for learning, portfolio, and professional demonstration purposes. Prompts should always be reviewed and adapted based on task requirements, audience, and ethical AI usage guidelines.
