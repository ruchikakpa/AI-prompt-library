# Productivity Prompts

## 1. Email Drafting Prompt

```text
Draft a professional email for the following purpose: [PURPOSE].

Audience: [RECIPIENT TYPE]
Tone: [TONE]
Length: [SHORT / MEDIUM / DETAILED]

Include:
- Clear opening
- Main message
- Required details
- Polite closing
```

## 2. Task Prioritization Prompt

```text
Help me prioritize the following tasks: [TASK LIST].

Consider:
- Urgency
- Importance
- Deadlines
- Effort required
- Dependencies

Provide:
- Priority order
- Reason for ranking
- Suggested schedule
```
